Leiam com atenção este README.

ao clonar o repositorio e abrir no seu pc, deem git pull origin main
Apos isso esperem baixar todos os arquivos da branch main.
Para mudar de branch veja se não mexeu em nenhum arquivo, pois vai da erro!!!
Comando para mudar de branch: git switch front 
após isso veja se tem alguem fazendo alguma alteração, se não tiver efetue o git pull origin front
